A simple API to generate random colors

Welcome to Colorand API, your ultimate solution for generating random colors for your projects. Whether you're a developer or a designer, our API provides you with an easy and reliable way to get a random color.

API Link

            https://colorand-api.com/api?type=hex
            Also you can replace "hex" with "rgb" or "rgba"

Fetch example

            fetch("https://colorand-api.com/api?type=hex")
  .then((res) => res.json())
  .then((data) => console.log(data));
